# Installation

!!! warning

	This page is a placeholder. Cadenzier has not yet reached an MVP (Minimum Viable Product) status.
	Estimations for when that might happen range from August 2026 to January 2027.

	Some installation methods might become available some time after reaching MVP status.

## Planned installation methods

### Linux

- nixpkgs
- AUR bin package
- Flatpak
- Binary executable

### macOS

- Homebrew
- Binary executable

### Windows

- Installer
- `.exe` executable

---

`cargo install` might be available, but highly discouraged for non-devs, as the install time and
bloat of Cargo could provide an awful UX.

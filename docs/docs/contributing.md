# Contributing

!!! warning

	This page is a placeholder. Cadenza has not yet reached an MVP (Minimum Viable Product) status.
	Estimations for when that might happen range from August 2026 to January 2027.

Cadenza is mainly a solo project, however, contributing is still possible. The main method of
contribution will be through donations, which I'll start accepting after Cadenza reaches MVP status.
Issues and Pull Requests on the Codeberg repo are also welcomed, but note that they may take some
time to get reviewed and merged. If you're not in the position for either of the two methods,
spreading the news about Cadenza is also a good way to contribute, as it might attract people who
can directly contribute.
